## 四、快速开始

### 环境配置
```bash
pip install -r requirements.txt
```

### 运行基础任务
```bash
python clustering_pipeline.py --data Tosches_turtle.h5ad --outdir results --epochs 50 --device cpu
```

### 运行创新方法 (多数据集)
```bash
python multi_dataset_runner.py
python comprehensive_analysis_report.py
```
